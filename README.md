***有你，才是一班。***

这是一个纪实类项目，我们打算在毕业那天把它作为一个惊喜送给大家，我们希望它能够留住初中生活的点滴美好回忆、友谊和事物。
换言之，我们打算写一部民间的《一史》，并且希望能够得到您的支持。我们将使用程序展现这段历史，如果你愿意帮忙，欢迎阅读 [文档](CONTRIBUTING.md) 并同我们一起撰写这部民间《一史》。
